Algunos de mis proyectos

Gestor de Reclamos Internos

Como proyecto grupal anual, desarrollamos un gestor de reclamos internos. Este gestor desarrollado en HTML, CSS, Angular y Django, permite a los empleados de una empresa registrar y gestionar sus reclamos internos de manera eficiente. El sistema incluye funcionalidades como la creación de reclamos, seguimiento del estado de los mismos y la posibilidad de calificar la resolución de los mismos.

Toolkit de Ubuntu

Como parte de un proyecto en grupo en la materia de Sistemas Operativos, desarrollamos un toolkit para Ubuntu. Programado en bash, este toolkit incluye una serie de herramientas y scripts que facilitan la administración del sistema operativo, como la gestión de usuarios, la monitorización del sistema y la optimización del rendimiento. El objetivo es proporcionar a los administradores de sistemas una herramienta útil y fácil de usar para mejorar la eficiencia en la gestión de sus sistemas.

Ver en GitHub

Broker de inversiones

Como parte de un proyecto en grupo en el módulo Programador I, desarrollamos un broker de inversiones. Programado en Python, este broker permite a los usuarios simular inversiones en diferentes acciones. El sistema incluye funcionalidades como la compra y venta de acciones, el seguimiento del rendimiento de las inversiones y la generación de informes. El objetivo es proporcionar a los usuarios una herramienta para aprender sobre inversiones y mejorar sus habilidades en el mercado financiero.

Ver en GitHub

Proyecto IoT Medicion de agua

Como parte de un proyecto en grupo en la materia de Aproximación al mundo del trabajo, desarrollamos un sistema IoT para medir el nivel de agua en un tanque. Este sistema utiliza sensores para medir el nivel de agua y enviar los datos a una aplicación web, programado en Micropython, una api en Flask, y datos en Grafana. El objetivo es proporcionar a los usuarios una herramienta para monitorizar el nivel de agua en tiempo real y recibir alertas cuando el nivel sea demasiado bajo.
